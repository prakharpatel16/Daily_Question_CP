# 🚀 Daily Coding Questions

Welcome to my **Daily Coding Questions** repository! This repository contains my daily solutions to programming problems from **LeetCode** and **Codeforces** as part of my journey to improve my problem-solving and Data Structures & Algorithms (DSA) skills.

## 📌 Goals

- Solve coding problems consistently every day.
- Strengthen DSA and algorithmic thinking.
- Prepare for coding interviews and competitive programming.
- Track progress and maintain coding discipline.


---

## 📖 Solution Format

Each solution includes:
- ✅ Problem Name
- 🔗 Problem Link
- 💡 Approach
- ⏱️ Time Complexity
- 📦 Space Complexity
- 💻 Code (C++)

Example:

```
Problem: Two Sum
Platform: LeetCode

Approach:
- Use a hash map to store previously visited numbers.
- Check if the complement exists.

Time Complexity: O(n)
Space Complexity: O(n)
```
